FishSTOC Cost Forecast Dashboard

What is this Model?

The FishSTOC Cost Forecast Dashboard predicts cloud costs for tenants (FMCs) using Oracle Cloud Infrastructure (OCI). It allows dynamic tenant addition with t-shirt sizing to estimate OCI utilization costs.

How it Works

Users add tenants, define their OCI usage with t-shirt sizes (Small, Medium, Large, XL), and set overall budget parameters. The dashboard generates forecasts using visualizations to analyze spend distribution, trends, and comparisons.

Inputs and Outputs

Inputs:

  • Annual Budget: Total budget allocated for cloud services.
  • Platform Admin Costs: Fixed monthly cost for managing the platform.
  • Tenants: Dynamically added tenants with t-shirt sizing.

Outputs:

  • Spend Distribution (Pie Chart): Displays spend by tenant.
  • Quarterly and Monthly Forecast (Column Chart): Shows spend over time.
  • Spend Trends (Line Chart): Illustrates actual vs. projected spend.
  • Budget Status (Gauge Chart): Compares spend to budget per tenant.
  • Tenant Spend Comparison (Bar Chart): Compares tenant spend in absolute values or percentages.
